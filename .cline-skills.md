## Common Issues & Solutions
- GeoDataFrame view vs copy: always use .copy()
- Fiona/GDAL errors: pin version 1.9.5
- psycopg2 errors: use SQLite only
- RecursionError: avoid custom wrappers
- Pandas pickles: use compatible versions

## Deployment Best Practices
- Pin all dependencies to stable versions
- Use SQLite for embedded database
- Single-file deployment for Streamlit Cloud
- Clear cache on schema changes
- Handle missing files gracefully
